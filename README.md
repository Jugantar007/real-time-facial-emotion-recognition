# Real-Time Facial Emotion Recognition

## Overview
This project implements a real-time facial emotion recognition system using CNN-based models trained for 4-class emotion detection: happy, sad, anger, and neutral.

## Features
- Real-time webcam-based emotion detection
- 4-class facial emotion classification
- OpenCV-based preprocessing and inference
- Edge deployment oriented workflow for Raspberry Pi

## Tech Stack
- Python
- OpenCV
- PyTorch / TensorFlow
- NumPy
